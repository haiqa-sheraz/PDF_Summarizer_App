# PDF Summarizer App

## Overview
PDF Summarizer App is a user-friendly application designed to summarize PDF documents using the LaMini-Flan-T5 model from Hugging Face. This model has 248M parameters and is utilized through the summarization pipeline provided by Hugging Face's Transformers library. The application is built with a user interface using Streamlit, allowing users to easily upload a PDF document and view its summary alongside the original document.

## Features
- Summarization of PDF documents using the LaMini-Flan-T5 model.
- Streamlit-based user interface for easy interaction.
- Side-by-side display of the original PDF document and its summary.
- Straightforward upload and summarization process.

## Model
The LaMini-Flan-T5 model from Hugging Face is used for summarization. With 248M parameters, this model is capable of generating concise and informative summaries of PDF documents.
Here is the link to model: https://huggingface.co/MBZUAI/LaMini-Flan-T5-248M

## UI Design
The user interface is built using Streamlit, a popular framework for creating web applications with Python. The design is intuitive and straightforward, allowing users to easily upload PDF documents and view their summaries.
